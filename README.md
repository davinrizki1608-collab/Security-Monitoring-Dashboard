# Security Monitoring Dashboard

Dashboard statis untuk monitoring keamanan: ringkasan alert kritis, aktivitas ancaman, risk posture, active incidents, dan live alert feed.

## Cara Menjalankan

Buka `index.html` langsung di browser.

## Struktur

- `index.html` - markup utama dashboard
- `styles.css` - styling responsif
- `app.js` - data demo, rendering tabel/feed, dan chart canvas

## Fitur

- Layout dashboard SOC responsif
- Sidebar navigasi dan topbar pencarian
- Metric cards untuk alert, event, login gagal, dan incident resolution
- Threat activity chart berbasis canvas
- Risk posture gauge dan progress per kategori
- Active incidents table
- Live alert feed
